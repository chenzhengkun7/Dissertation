Background

Time-Domain Statistics

  • First moment (mean):
    $M^{k} = \frac{1}{n} \sum\limits_{i=1}^n X_i^k$
  • Second moment (variance):
    $\sigma^2 = \frac{1}{(n-1)} \sum\limits_{i=1}^n (X_i-\bar{X})^2$
  • Third moment:
    $\mathrm{Skewness} = \frac{M^{3}}{\sigma^3}$
  • Fourth moment:
    $\mathrm{Kurtosis} = \frac{M^{4}}{\sigma^4}$

Background

Spectral Power Approximation

\begin{equation} X[n] = \sum_{k=0}^{N-1} x[k] e^{-j \frac{2 \pi}{N} n k} \end{equation}

Conclusion

Research Questions

  • Developed a data acquisition and transmission strategy for machine health monitoring
  • Presented a case study which used Scikit-Learn to predict tool health
  • Showed that embedded computing can be used for:
    • Vibration data acquisition
    • Feature extraction and preprocessing
    • Machine learning inference
  • Embedded system performance on par with cloud computing

Conclusion

Limitations

  • Feasibility demonstrated for single machine
  • Generalized vibration analysis exclusively used. Unique considerations are likely required for other applications.
    • For example, can similar feature extraction measures be taken on image data?
  • Long-term data storage and accessibility are not directly addressed here

Conclusion

Future Work

  • Continue to investigate feasibility of embedded computing for true real-time edge inference
  • Build edge processing strategies for other large data types
    • Images
    • Audio
  • Standardize health monitoring messaging syntax
