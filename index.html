Model Validation

  • Repeated March experiments
  • Ran existing model (v1) while performing data acquisition
    • Generate full dataset considering machine downtime (3 months), new DAQ
    • Test model (v1)
  • Developed updated model (v2) based on these new data
  • Validated this model on (v2) unseen parameters

Model Performance (v1)

  • Null Hypothesis: tool is healthy
  • Balanced accuracy: 85.4%
  • True Positive Rate: 94.6%
  • True Negative Rate: 76.2%
  • Precision: 0.700
  • Recall: 0.946
  • This is on parameters that have been seen before

Model Performance (v2)

  • Balanced accuracy: 88.4%
  • True Positive Rate: 93.2%
  • True Negative Rate: 83.6%
  • Precision: 0.820
  • Recall: 0.932
